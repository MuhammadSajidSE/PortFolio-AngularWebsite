Final Year Project (FYDP)

With the rise of cloud and containerized applications, managing security has become increasingly challenging. Auto-scalable indirect DDoS attacks exploit cloud auto-scaling to exhaust resources and disrupt services.

Our project introduces a framework that integrates Intent-Based Networking (IBN) with Kubernetes orchestration to proactively monitor traffic, enforce security policies, and dynamically respond to threats.

This approach strengthens resilience in cloud-native systems, ensuring secure, reliable, and scalable applications.
